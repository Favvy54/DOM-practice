# Week 7: DOM & Events

This repository contains my solutions for the week 7 assignment of the Upskill Web Development Program, covering DOM manipulation and event handling

## Tasks

### 1. DOM Explorer (task1.html)
An HTML Page with a heading, two paragraphs, and a button
- Select and log each element to the console
- Update the heading text when the page loads
- Add a new paragraph when the button is clicked

### 2. Interactive To-Do List (task2.html)
A to-do list app built with vanilla JavaScript
- Add tasks via a text input and Add button
- Delete tasks with a Delete button
- Mark tasks as completed by clicking them (toggles a CSS class)

### 3. Event Propagation Demo (task3.html)
Demonstrates event bubbling using a nested '<div>' structure, with click listeners on each level logging which element was clicked. Includes a button that calls `stopPropagation()` to show how it changes bubbling behaviour.

## Tech
Vanilla JavaScript, HTML, and CSS
